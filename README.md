# MKDX-English-Mod

This mod will translate Mario Kart Arcade GP DX from Japanese to English.
Note: This mod is still very much a work in progress!

## Installing

* Make sure you have MKDX version 1.10.22!
* Download latest release
* Unzip
* Run the patcher
